Un texte avec Cormorant Garamond, qui fonctionne aussi en italique.

Le corps de texte utilise IBM Plex Sans en 16px avec une hauteur de ligne de 1.75 pour une lisibilité optimale. Les paragraphes sont naturellement contraints à 65ch de largeur maximale — le sweet spot pour la lecture sur écran.

Les liens sont clairement identifiables grâce à leur couleur primaire. Le texte peut être en gras, en italique, surligné, barré, en sub, sup ou rendu en petite taille.

Les abréviations comme HTML et CSS sont marquées avec un soulignement pointillé. Le code inline ressemble à display: block et les raccourcis clavier à Ctrl + K.

Bloc de code

/* Variables CSS — Personnalisation en un endroit */
:root {
  --color-primary: #0f62fe;
  --font-sans:     'IBM Plex Sans', sans-serif;
  --space-5:       1rem;
}

/* Résultat : HTML sémantique, beau sans classes */
<article>
  <h1>Titre</h1>
  <p>Contenu…</p>
</article>

Accordéons (details/summary)

Comment personnaliser les couleurs ?

Surchargez simplement les variables CSS dans votre propre feuille de style. Par exemple : :root { --color-primary: #your-color; } Toutes les couleurs se mettront à jour automatiquement.

Comment ajouter un thème sombre ?

Ajoutez un bloc @media (prefers-color-scheme: dark) et redéfinissez les variables de couleur. Les tokens sont déjà prêts à être surchargés : 

@media (prefers-color-scheme: dark) {
                :root {
                    --color-bg:   #161616;
                    --color-text: #f4f4f4;
                    /* … autres tokens … */
                }
                }
Y a-t-il des dépendances JavaScript ?

Aucune. Ce framework est 100% CSS. Aucun JS n'est nécessaire pour le style de base. Les interactions natives du navigateur (<details>, <dialog>) fonctionnent sans script.
